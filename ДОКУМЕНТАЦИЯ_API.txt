Базовый URL:
http://127.0.0.1:8000/api/v1/

1. Получить список курсов
URL: /courses/
Метод: GET
Описание: Возвращает список всех доступных для покупки курсов.
Параметры запроса: Нет

2. Получить информацию о курсе
URL: /courses/{id}/
Метод: GET
Описание: Возвращает информацию о конкретном курсе по его ID.
Параметры запроса:
id (integer) ID курса

3. Купить курс
URL: /courses/{id}/pay/
Метод: POST
Описание: Позволяет пользователю приобрести курс. При успешной покупке пользователь подписывается на курс и получает доступ.
Параметры запроса:
id (integer) ID курса

4. Получить список уроков для курса
URL: /courses/{course_id}/lessons/
Метод: GET
Описание: Возвращает список всех уроков для указанного курса.
Параметры запроса:
course_id (integer) ID курса

5. Получить информацию о конкретном уроке
URL: /courses/{course_id}/lessons/{lesson_id}/
Метод: GET
Описание: Возвращает информацию о конкретном уроке по его ID.
Параметры запроса:
course_id (integer) ID курса
lesson_id (integer) ID урока

6. Получить список пользователей
URL: /users/
Метод: GET
Описание: Возвращает список всех пользователей.
Параметры запроса: Нет


Примечания
Аутентификация: Для доступа к методам, связанным с покупкой курсов и подписками, требуется аутентификация (Djoser).
Ошибки: В случае ошибок, таких как недостаток средств или недоступность курса, API возвращает соответствующие сообщения об ошибках с кодом статуса HTTP.
